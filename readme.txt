# 単眼男子
単眼の男の子と仲良くなりましょう。

## 遊び方
- 一緒にいる時間や会った回数が増えると、仲良くなっていきます。
  - 条件を満たすと、メニューに「距離を縮める」が出現します。
  - 「距離を縮める」を選ぶと、ぐっと距離が縮まる…かも。
- スキンシップやコミュニケートは親密度に関わりません。
- コミュニケートの話題は日替わりだったりします。

## 触れる箇所
頭、顔、耳、胸、肩
- つつくとその場所に注目した感じになったりします。
- 肩をつつくと「何か話して」になります。
- その他の箇所をつつくとメニューを開きます。

## 季節イベント
ちょっとある
