# 単眼男子
単眼の男の子と仲良くなりましょう。
ユーザは男性固定で、喋るのでご注意を。

## 遊び方
- たまに重要っぽいトークが発生します。
- 条件を満たすと、メニューに「ランクアップ」が出現します。
- 「ランクアップ」を選ぶと、ぐっと距離が縮まる…かも。
- スキンシップは親密度に関わりません。好きなだけ触れます。

## 触れる箇所
頭、顔、耳、胸、肩
- 肩をつつくと「何か話して」になります。
- その他の箇所をつつくとメニューを開きます。

## ショートカット
[m]メニュー
[t]何か話して
[c]話しかける
