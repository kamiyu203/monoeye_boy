# 単眼男子
単眼の男の子と仲良くなりましょう。
ユーザの性別は選べますが、ちょっと喋るのでご注意を。

## 遊び方
- 一緒にいる時間や会った回数が増えると、仲良くなっていきます。
- 条件を満たすと、メニューに「ランクアップ」が出現します。
- 「ランクアップ」を選ぶと、ぐっと距離が縮まる…かも。
- スキンシップやコミュニケートは親密度に関わりません。

## 触れる箇所
頭、顔、耳、胸、肩
- 肩をつつくと「何か話して」になります。
- その他の箇所をつつくとメニューを開きます。

## ショートカット
[M]メニュー
[T]何か話して
[C]話しかける(少し仲良くなってから)
